AI Outbound Calling Tool Data Analysis

Extracting script and text insights from large-scale conversational data.

Project Overview

Managed and analyzed data for an AI-driven outbound calling tool. With a dataset exceeding 40,000 records, I leveraged Python and Machine Learning to uncover deep insights into script performance and conversational patterns.

Key Contributions

  • Performed end-to-end Data Extraction, EDA, Transformation, and Cleaning.
  • Designed critical KPIs to measure AI calling effectiveness and conversion rates.
  • Implemented ML-based LDA (Latent Dirichlet Allocation) classification for text insights.
  • Developed data models and DAX calculations for advanced visualization.

Business Impact

The analysis provided the business with a clear understanding of which scripts were most effective, leading to optimized AI conversational flows and improved outbound performance.

Technologies Used

Python Scikit-learn LDA Power BI

Focus Area

Data Scale
